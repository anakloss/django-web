=== COMANDOS ===
Crear proyecto:
    django-admin startproject [nombre]
    python manage.py startapp [nombre_app]

Iniciar BD:
    python manage.py makemigrations
    python manage.py migrate

Generar SQL:
    python manage.py sqlmigrate [nombre_app] [num migracion]

Ejecutar servidor:
    python manage.py runserver

Check errores:
    python manage.py check [nombre_app]

Crear super usuario:
python manage.py createsuperuser


=== SHELL ===
python manage.py shell
*** Insertar en BD:
    >>> from GestionPedidos.models import Articulo
    Forma 1
    >>> art=Articulo(nombre='Mesa', seccion='Deco', precio=5000)
    >>> art.save()
    Forma 2
    >>> art2=Articulo.objects.create(nombre='Silla', seccion='Deco', precio=8500)

*** Actualizar en BD:
    >>> art.precio=5800
    >> art.save()

*** Borrar en BD:
    >>> art3=Articulo.objects.get(nombre='Silla')
    >>> art.delete()

*** Consultar en BD:
    >>> lista=Articulo.objects.all()
    [>>> lista.query.__str__()]


=== Config para PostgreSQL ===
1. pip install psycopg2
2. settings.py -> DATABASES
    'ENGINE': 'django.db.backends.sqlite3',
    'NAME': [nombre_DB],
    'USER': dbuser,
    'PASSWORD': dbpass,
    'HOST': '127.0.0.1',
    'DATABASE_PORT': '5432',
3. Iniciar BD (*)
